# PRD: 문홍배 퍼스널 브랜딩 랜딩페이지 v2

\---

## 1\. 목적

이 문서는 금융 컨설턴트 문홍배의 퍼스널 브랜딩 랜딩페이지를 Antigravity로 생성하기 위한 요구사항을 정의한다.

* 방문자는 페이지를 열었을 때 10초 이내에 "누구인지 / 무엇을 하는지 / 어떻게 연락하는지"를 파악할 수 있어야 한다.
* 페이지는 단일 HTML 파일로 완성되며, 서버나 데이터베이스를 사용하지 않는다.

\---

## 2\. 포함할 것 (In Scope)

* \[ ] Hero 섹션: 이름(한글·영문), 직함 2개, 슬로건 1줄, CTA 버튼 2개(이메일·전화)를 표시한다.
* \[ ] Services 섹션: 컨설팅 서비스 3종을 카드로 표시한다. Hero 다음에 바로 위치한다.
* \[ ] Stats 섹션: 숫자 카드 2개(35년 금융경력 / 17년 M\&A·전략·구조조정)를 표시한다. Services 다음에 위치한다.
* \[ ] Stats 카드1의 숫자(35년)는 font-size 26px로 표시하고, 카드2의 숫자(17년)는 font-size 20px로 표시한다. 카드2의 숫자가 카드1보다 작은 이유는 35년 중 17년임을 시각적으로 나타내기 위함이다.
* \[ ] Career 섹션: 산업은행 커리어 4개 항목을 카드 그리드로 표시한다. 카드 순서는 기업금융 → 조사분석 → M\&A·전략·구조조정→ 기타  순이다.
* \[ ] Track Record 섹션: 주요 실적 4개를 태그+설명 형태의 리스트로 표시한다.
* \[ ] Activities 섹션: 대외활동을 현재(現) / 이전(前)으로 분리하여 표시한다.
* \[ ] Education 섹션: 학력 3개와 자격증 3개를 표시한다.
* \[ ] Contact 섹션: 이메일 버튼(mailto:)과 전화 버튼(tel:)을 나란히 표시한다.
* \[ ] 페이지는 뷰포트 너비 375px(모바일)\~1280px(데스크탑)에서 레이아웃이 깨지지 않는다.
* \[ ] 한국어 본문 폰트는 Noto Sans KR, 제목 폰트는 Noto Serif KR을 사용한다.
* \[ ] Hero 섹션 아바타 자리에 프로필 사진을 원형(border-radius: 50%)으로 표시한다. 크기는 width=80px, height=80px이며, object-fit=cover, object-position=center top을 적용한다.
* \[ ] 프로필 사진은 Base64로 인코딩하여 HTML 파일 내부에 직접 내장한다. 별도의 이미지 파일을 사용하지 않는다.

\---

## 3\. 제외할 것 (Out of Scope)

* \[ ] Stats 카드 3번째 항목(11년 M\&A·구조조정 자문)은 표시하지 않는다.
* \[ ] 블로그, 뉴스, 아티클 섹션은 만들지 않는다.
* \[ ] 문의 폼(form submit)과 서버 전송 기능은 구현하지 않는다.
* \[ ] 관리자 대시보드, CMS, 로그인 기능은 포함하지 않는다.
* \[ ] 영어 다국어 버전은 만들지 않는다.
* \[ ] 애니메이션은 페이지 로드 시 fade-in 1회에 한정하며, 반복 루프 애니메이션은 사용하지 않는다.
* \[ ] Hero 아바타 자리에 글자("문") 텍스트 원형은 사용하지 않는다. 반드시 실제 프로필 사진을 표시한다.

## 5\. 완성 체크리스트

생성된 결과물이 아래 항목을 모두 만족해야 완성으로 간주한다.

### 콘텐츠

* \[ ] Hero 섹션에 이름(한글·영문), 직함 2개, 슬로건이 모두 표시된다.
* \[ ] Hero 섹션 아바타 자리에 프로필 사진이 원형으로 표시되며, 글자 아바타("문")는 없다.
* \[ ] Services 섹션이 Hero 바로 다음에 위치한다.
* \[ ] Services 카드가 3개이고, 서비스2 설명은 "투자, 융자, 정책자금, 수출", 서비스3 제목은 "창업컨설팅 기타", 설명은 "사업계획서, 법인 설립, 인증, 소상공인"이다.
* \[ ] Stats 섹션이 Services 다음에 위치한다.
* \[ ] Stats 카드가 2개이고 숫자는 35년·17년만 표시된다. 11년 카드는 없다.
* \[ ] Stats 카드1 숫자(35년)의 font-size(26px)가 카드2 숫자(17년)의 font-size(20px)보다 크다.
* \[ ] Career 카드가 4개이고 순서가 기업금융 → 조사분석 → M\&A·전략·구조조정 → 기타  이다.
* \[ ] Track Record 항목이 4개이고 태그가 분리되어 있다.
* \[ ] 대외활동이 현재(現)와 이전(前)으로 구분되어 표시된다.
* \[ ] 학력 3개와 자격증 3개가 표시된다.
* \[ ] 이메일(hbmoon21@gmail.com)과 전화(010-2098-9358)가 정확히 표시된다.

### 기능

* \[ ] 이메일 버튼 클릭 시 `mailto:hbmoon21@gmail.com`이 실행된다.
* \[ ] 전화 버튼 클릭 시 `tel:010-2098-9358`이 실행된다.
* \[ ] 375px 뷰포트에서 가로 스크롤이 발생하지 않는다.
* \[ ] 1280px 뷰포트에서 콘텐츠가 중앙 정렬되어 표시된다.

### 디자인

* \[ ] 제목에 Noto Serif KR이 적용되어 있다.
* \[ ] 본문에 Noto Sans KR이 적용되어 있다.
* \[ ] 페이지 전체에 gradient, 과도한 그림자 없이 flat 디자인이 적용된다.
* \[ ] 각 섹션 사이에 구분선 또는 충분한 여백(최소 24px)이 있다.

### 기술

* \[ ] 파일이 단일 HTML 파일 1개로 완성된다.
* \[ ] 프로필 사진이 Base64로 HTML 내부에 내장되어 있으며, 외부 이미지 파일 경로를 참조하지 않는다.
* \[ ] 외부 의존성이 Google Fonts CDN 외에 없다.
* \[ ] `<form>` 태그를 사용하지 않는다.
